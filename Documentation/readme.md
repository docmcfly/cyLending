# Documentation

Mit dieser Erweiterung kann man Objekte an andere Ausleihen.

## Features

* Kalender mit Übersicht der ausgeliehen Objekte bzw. der Objekte für die eine Ausleihanfrage voliegt. 
* Objekte können automatisch ausgeliehen werden oder man muss erst einen Genehmigungsprocess durchlaufen
* Man hat immer den Überblick über seine Ausleihen und seine Ausleihanfragen
* Ausleihgenehmiger und andere Personen können über den Ausleihprocess via Mail informiert werden
* In Genehmigungs
